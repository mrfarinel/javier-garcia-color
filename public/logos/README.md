# Official Logo Assets

Place only original official client/platform logo files in this folder.

Rules:

- Do not recreate logos.
- Do not trace or vectorize screenshots.
- Do not use Wikipedia, Logos-World, Seeklogo, SVGRepo, Brands of the World, or similar repositories.
- Use official assets from the company's website, press kit, or brand center.
- Prefer SVG.
- If SVG is unavailable, use official PDF/EPS/PNG with transparent background.
- If only an official black logo is available, convert that official file to pure white without changing geometry.
